# JavaScript String Tasks

Решение задач по работе со строками в JavaScript.

## Описание задач

### Задача 1: concatenateStrings
Функция для конкатенации двух строк с удалением всех пробелов.

### Задача 2: getStringLength
Функция для получения длины строки.

### Задача 3: getStringFromTemplate
Функция для создания приветствия по шаблону с использованием интерполяции строк.

### Задача 4: getChar
Функция для получения символа по порядковому номеру.

### Задача 5: removeFirstOccurrences
Функция для удаления первого вхождения подстроки.

### Задача 6: getRectangleString
Функция для создания строкового представления прямоугольника.

## Запуск тестов

```bash
node test.js
```

## Файлы

- `string-tasks.js` - реализация всех функций
- `test.js` - автоматические тесты для проверки корректности работы
